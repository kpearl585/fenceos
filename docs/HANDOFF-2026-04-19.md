## Critical codebase context (learned the hard way)

- **Two parallel estimating engines exist:** `fence_graphs` (used by Advanced Estimator + quote portal) and `fence_designs` (used by Phase 1 estimator). The AR feature uses `fence_graphs` only. Don't confuse them.
- **The quote portal is at `/quote/[token]`** (customer-facing, unauthenticated, token-validated via `is_token_valid()` RPC)
- **Fence type is derived from `input_json.productLineId`** via substring match — the `derive_ar_fence_type_id()` Postgres function is authoritative. MVP only matches wood+6ft.
- **Design system:** fence-palette (`bg-fence-950`, `bg-fence-600` primary, white cards on dark blue). NOT the dark `#080808` landing-page theme. Match patterns from `src/components/ShareQuoteButton.tsx` when building new UI.
- **Supabase clients:** `createClient()` from `@/lib/supabase/server` (RLS-aware via cookies), `createAdminClient()` (service role, bypasses RLS). Use admin client after validating a token.
- **Rate limiter pattern:** `RateLimiters.opName(key)` in `src/lib/security/rate-limit.ts` returns `{ success, error, remaining }`. Add new limiters there.
- **Pre-commit hook runs `npm run build`** — blocks commits on TS errors. Don't fight it. Build must pass.
- **Supabase CLI, Vercel CLI, gh CLI all installed and authenticated locally.**

### Gotchas already resolved (don't re-solve)

- Vercel preview env vars were only in production scope → `MIDDLEWARE_INVOCATION_FAILED` on preview deploys. Fixed by adding `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_ROLE_KEY` to preview for the `claude/elated-jones-e92a23` branch. New feature branches may need the same fix.
- Sketchfab asset was misleadingly titled "CC0" but actual license is CC BY 4.0. Attribution is required, not optional.
- `derive_ar_fence_type_id()` had a search_path mutability warning — already fixed with `ALTER FUNCTION ... SET search_path = public, pg_catalog`.
- Next.js 16 middleware runs at runtime (not edge-only), so env vars must be set in the Vercel runtime env, not just at build time.
